## Built With

* [React](https://pl.reactjs.org/)

* [Material-UI](https://material-ui.com/)

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AlexVilbitskyi/HexOcean-recruitment-exercise.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run app
   ```sh
   npm start
   ```
